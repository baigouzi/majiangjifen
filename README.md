# 麻将计分器

一个简单的在线麻将计分工具，用于记录和管理麻将游戏的分数。

## 功能特点

- 记录玩家昵称和分数
- 支持多轮游戏计分
- 自动计算总分
- 保存游戏数据到GitHub

## 使用方法

1. 访问页面
2. 输入玩家昵称
3. 记录每轮游戏的分数
4. 查看总分统计

## 数据保存

游戏数据会自动保存到GitHub仓库，需要配置GitHub Token才能使用此功能。

## 技术实现

- HTML5 + CSS3 + JavaScript
- GitHub API 用于数据存储
- GitHub Actions 实现自动构建和Token注入
